# QR BTF
这个项目叫 [QR-BTF](https://qrbtf.com/) :left_point:点击进入官网。
网站风格也很有意思。

其实很早就有生成个性化二维码的产品了： [Go!](https://classic.qrbtf.com/)

根据README介绍安装后：
```
git clone https://github.com/ciaochaos/qrbtf.git
cd qrbtf
npm install
npm audit fix --force
npm start
```
结果最终效果竟然是它的本地的官网： [Go!](https://classic.qrbtf.com/)
好吧，感兴趣的去体验一下。
